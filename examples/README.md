# Examples

Примеры использования библиотеки `@ojson/models`.

## todo-api

Полноценное Express API для управления todo-lists с использованием models и TypeScript. Демонстрирует:

- Интеграцию `withModels` с Express.js
- Создание моделей для CRUD операций
- Автоматическую мемоизацию результатов
- TypeScript типизацию моделей и API

См. [todo-api/README.md](./todo-api/README.md) для подробностей.

## deadline-interruption

Демонстрирует, как преобразование async/await в генераторы через `esm-loader` позволяет `withDeadline` прерывать выполнение на любом шаге, а не только в конце.

См. [deadline-interruption/README.md](./deadline-interruption/README.md) для подробностей.
